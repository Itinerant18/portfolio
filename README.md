# Portfolio Website

A modern, interactive portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Features
- Interactive terminal interface
- Project showcase with detailed views
- Skills and experience timeline
- AI-powered sidebar
- Code editor theme

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase
- Framer Motion
